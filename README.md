# Boilerplates

A personal collection of production-ready boilerplates for kickstarting new projects. Each one is self-contained, opinionated, and designed to eliminate repetitive setup so you can focus on building features from day one.

## Available Boilerplates

| Name | Stack | Description |
|------|-------|-------------|
| [express-api](./express-api/) | Express · TypeScript · PostgreSQL · Docker | Modular REST API with JWT auth |

## Usage

Each boilerplate lives in its own directory and has a dedicated README with setup instructions. Clone the repo and copy whichever boilerplate you need:

```bash
git clone https://github.com/pedrohhope/boilerplates.git
cp -r boilerplates/express-api my-new-project
cd my-new-project
```
